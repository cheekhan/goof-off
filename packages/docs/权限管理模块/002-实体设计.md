# 权限管理模块

## 1、角色实体

| 字段     | 说明               |
| -------- | ------------------ |
| roleID   | 角色的id           |
| name     | 角色名称           |
| extendID | 继承的某个低级角色 |
| remark   | 角色描述           |


## 2、用户实体


| 字段     | 说明                         |
| -------- | ---------------------------- |
| userID   | 用户唯一id                   |
| name     | 用户名                       |
| acount   | 登录账号                     |
| roleDTO  | 和用户关联的角色信息，是列表 |
| groupDTO | 用户关联了的用户组，是列表   |


## 3、用户组
> 有层级关系

| 字段     | 说明                         |
| -------- | ---------------------------- |
| groupID  | 用户组id                     |
| name     | 用户组名称                   |
| roleDTO  | 用户组关联的角色信息，是列表 |
| parentID | 给用户组形成嵌套关系         |
